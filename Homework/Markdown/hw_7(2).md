### 2. 令 $U = \{(x_1, x_2, x_3, x_4) \mid x_1 + 2x_2 + 3x_3 - x_4 = 0\}$. 它是行向量空间 $\mathbb{R}^4$ 的子空间. 

#### (1) 证明 $u_1 = (2, -1, 0, 0)$, $u_2 = (3, 0, -1, 0)$, $u_3 = (1, 0, 0, 1)$ 是 $U$ 的一个基. 

**解：**
$u_1, u_2, u_3 \in U$

设 $a u_1 + b u_2 + c u_3 = 0$

$$
\begin{cases}
2a + 3b + c = 0 \\
-a = 0 \\
-b = 0 \\
c = 0
\end{cases}
$$

$a = b = c = 0$, $u_1, u_2, u_3$ 线性无关，故为 $U$ 的一个基



#### (2) 证明 $v_1 = (1, 1, -1, 0)$, $v_2 = (1, 1, 0, 3)$, $v_3 = (1, 0, 1, 4)$ 是 $U$ 的一个基.

**解：**

$v_1, v_2, v_3 \in U$

$a v_1 + b v_2 + c v_3 = 0$

$$
\begin{cases}
a + b + c = 0 \\
a + b = 0 \\
-a + c = 0 \\
3b + 4c = 0
\end{cases}
$$

$a = b = c = 0$, $v_1, v_2, v_3$ 线性无关，故为 $U$ 的一个基

### 5. 设 $U$ 和 $V$ 是 $\mathbb{R}^n$ 的子空间，$\dim(U + V) = 1 + \dim U \cap V$. 证明：$U + V$ 是这两个子空间中的一个，$U \cap V$ 是这两个子空间中的另一个.

**解：**设 $\dim(U \cap V) = a$, $\dim U = a + t$, $\dim V = a + s$
$$
\dim(U + V) = \dim U + \dim V - \dim(U \cap V)
$$

$$
= 1 + \dim(U \cap V)
$$

$$
a + s + a + t - a = 1 + a
$$

$$
s + t = 1
$$
故 $s = 1, t = 0$ 或 $s = 0, t = 1$

若 $s = 1, t = 0$, $\dim U = a$, $\dim V = a + 1$

$U \subset V$

故 $U = U \cap V$, $V = U + V$

若 $s = 0, t = 1$, 同理, $U = U + V$, $V = U \cap V$



### 6. 证明：子空间 $U_1, U_2, \ldots, U_k$ 线性无关当且仅当
$$
(U_1 + U_2 + \cdots + U_{i-1}) \cap U_i = 0, \quad 1 < i \leq k.
$$
**解：**先证充分性：$\Rightarrow$  

当 $(U_1 + \cdots + U_{i-1}) \cap U_i = 0$，$(1 < i \leq k)$  
取各自任意向量，s.t. $u_1 + \cdots + u_k = 0$  
$$
-u_k = u_1 + \cdots + u_{k-1} \in U_k \cap (U_1 + \cdots + U_{k-1})
$$
故 $u_k = 0$，同理，$u_1 = u_2 = \cdots = u_k = 0$  

故线性无关  

再证必要性：$\Leftarrow$  

若线性无关  
设 $x \in (U_1 + \cdots + U_{i-1}) \cap U_i$  

令 $x = u_i = u_1 + \cdots + u_{i-1}$  
$$
u_1 + \cdots + u_{i-1} - u_i = 0
$$
故  
$$
u_1 = \cdots = u_{i-1} = u_i = 0
$$
故  
$$
(U_1 + \cdots + U_{i-1}) \cap U_i = 0
$$

### 7. 设 $U, V, W$ 是 $\mathbb{R}^n$ 的子空间. 下列等式是否成立？
$$
U \cap (V + W) = (U \cap V) + (U \cap W).
$$
**解：**

7、不成立  
$$
U = \{(x,y) \mid x = y\}
$$
$$
V = \{(x,0) \mid x \in \mathbb{R}\}
$$
$$
W = \{(0,y) \mid y \in \mathbb{R}\}
$$
$$
U \cap (V + W) = U = \{(x,y) \mid x = y\}
$$
$$
(U \cap V) + (U \cap W) = \{0\}
$$

### 10. 设 $u = (1, 1, 1, 0)$, $v = (0, 1, 1, 1)$, $w = (1, 1, 0, 0)$ 是 $\mathbb{R}^4$ 中的向量. 
#### (2) 证明不存在纯量 $a, b, c$ 使得 $a u + b v + c w = (1, 2, 3, 4)$. 
**解：**
$$
\left\{ 
\begin{array}{l} 
a + c = 1 \\ 
a + b + c = 2 \\ 
a + b = 3 \\ 
b = 4 
\end{array} 
\right.
$$

无解，故不存在
