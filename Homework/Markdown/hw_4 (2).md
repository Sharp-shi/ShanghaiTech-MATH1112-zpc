## 习题2.3

### 4. 对任意的 $m \times s$ 矩阵 $A$ 和 $s \times n$ 矩阵 $B$，证明：$\text{rank } A + \text{rank } B - s \leq \text{rank}(AB)$

**解：**构建分块矩阵  
$M = \begin{pmatrix} AB & 0 \\ 0 & E_s \end{pmatrix}$  

$\text{rank } M = \text{rank } (AB) + s$  

对 $M$ 进行初等行变换  
$M = \begin{pmatrix} AB & 0 \\ 0 & E_s \end{pmatrix} \longrightarrow \begin{pmatrix} AB & 0 \\ -B & E_s \end{pmatrix} \longrightarrow \begin{pmatrix} 0 & A \\ -B & E_s \end{pmatrix}$  

故  
$\text{rank } M \geq \text{rank } \begin{pmatrix} 0 & A \\ -B & E_s \end{pmatrix} = \text{rank } A + \text{rank } B$  

$\text{rank } A + \text{rank } B - s \leq \text{rank } (AB)$



### 5. 证明：如果三个 $n$ 阶方阵的乘积为 $0$，那么它们的秩的和不超过 $2n$

**解：**令 $ABC = 0$

由习题4可知  
$\text{rank } AB + \text{rank } C - n \leq \text{rank } ABC = 0$

$\text{rank } AB + \text{rank } C \leq n$

又 $\text{rank } A + \text{rank } B - n \leq \text{rank } AB$

故 $\text{rank } A + \text{rank } B + \text{rank } C \leq 2n$

### 8. 设 $A$ 和 $B$ 是方阵.证明如果 $E + AB$ 可逆，那么 $E + BA$ 可逆.

**解：**令$X = E - B(E + AB)^{-1}A$

$X(E + BA) = E + BA - B(E + AB)^{-1}A - B(E + AB)^{-1}ABA$

又 $\because$ $(E + AB)^{-1}(E + AB) = E$

$(E + AB)^{-1}AB = E - (E + AB)^{-1}$

$(E + AB)^{-1}ABA = A - (E + AB)^{-1}A$

故 $X(E + BA) = E + BA - B \cdot A = E$

故 $X = (E + BA)^{-1}$

故 $E + BA$ 可逆，且 $(E + BA)^{-1} = E - B(E + AB)^{-1}A$
